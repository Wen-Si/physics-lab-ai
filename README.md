# 物理实验AI智能体

一个能够通过自然语言模拟物理实验的AI智能体，支持3D动态渲染和交互式查看。

## 功能特点

- **自然语言输入**: 用户可以用自然语言描述物理实验
- **智能理解**: 使用智谱GLM-4.5-Flash大模型进行自然语言理解
- **12节点工作流**: 完整的实验生成流程，包括：
  1. 输入解析节点
  2. 意图识别节点
  3. 实验类型分类节点
  4. 物理参数提取节点
  5. 物理定律匹配节点
  6. 实验场景构建节点
  7. 物理计算节点
  8. 3D模型生成节点
  9. 动画序列生成节点
  10. 文字描述生成节点
  11. 结果验证节点
  12. 输出格式化节点
- **3D可视化**: 使用Three.js进行实时3D渲染
- **交互式控制**: 支持鼠标旋转、缩放、平移查看3D场景
- **物理定律应用**: 自动匹配和应用相关物理定律

## 支持的实验类型

- **力学**: 自由落体、单摆、弹簧振子、碰撞、斜面运动
- **电磁学**: 电磁感应、电路分析、电荷运动
- **光学**: 光的折射、透镜成像、光的干涉
- **热力学**: 热传导、理想气体状态变化
- **波动**: 波的传播、多普勒效应
- **现代物理**: 粒子运动、量子效应

## 技术栈

- **前端**: React 18, Next.js 14, TypeScript
- **3D渲染**: Three.js, React Three Fiber, React Three Drei
- **AI模型**: 智谱GLM-4.5-Flash
- **工作流引擎**: 自研12节点工作流系统

## 本地运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build

# 部署到GitHub Pages
npm run deploy
```

## 使用示例

1. 输入实验描述，例如："模拟一个质量为2kg的小球从10米高处自由落下的过程"
2. 点击"开始模拟"按钮
3. 查看3D渲染结果，可以用鼠标旋转查看
4. 阅读详细的实验解释和物理定律说明
5. 使用控制面板调整动画速度和参数
